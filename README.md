# bfrb-detection
BFRB Detection using Deep Learning and NLP

## How to get the data

The files `data/train.csv` and `data/test.csv` are not included in version control.
To obtain these files, please visit the Kaggle competition page:

https://www.kaggle.com/competitions/cmi-detect-behavior-with-sensor-data/data

Download the required files and place them in the `data/` folder.
